# Superstore Sales Analysis Using SQL

## Description
Superstore sales analysis using SQL, CTEs, Subqueries, Window Functions, and Joins.

---

## Author
**Shauryansh Chauhan**

---

## Project Overview
This project analyzes the Superstore dataset using MySQL Workbench to extract meaningful business insights such as customer performance, sales trends, and rankings.

---

## Dataset
The dataset contains information about:
- Customers
- Orders
- Products
- Sales
- Profit
- Quantity

Raw data is stored in `superstore_raw`.

---

## Database Structure

The dataset is divided into three tables:

### Customers
Stores unique customer details.

### Products
Stores product-related information.

### Orders
Stores transaction details including sales and profit.

---

## SQL Concepts Used

- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- Joins
- Aggregations

---

## Key Analysis Performed

- Orders with sales above average
- Highest sales order per customer
- Total sales per customer
- Customers above average sales
- Customer ranking based on sales
- Row numbering per customer orders
- Top and bottom customers

---

## Final Insights

This project answers:
- Who are the top 5 customers?
- Who are the bottom 5 customers?
- Which customers made only one order?
- Which customers have above-average sales?
- What is the highest order value per customer?

## Tools Used

- MySQL Workbench
- SQL

This project demonstrates how SQL can be used to transform raw data into meaningful business insights using advanced querying techniques.
